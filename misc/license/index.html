授權

昭源字體以 SIL OPEN FONT LICENSE Version 1.1（SIL 開放字型授權版本 1.1，簡稱 SIL OFL 或 OFL）條款授權。

這個授權的自由度甚高，可將字型作以下的個人或商業用途，費用全免：

  • 圖片或影片製作
  • 書籍出版
  • PDF 製作時嵌入字體子集（不必轉為曲線）
  • 網頁使用
  • 作業系統、電子裝置內嵌（例如機頂盒、閲讀器、遊戲主機）
  • 附載於其他軟件或應用程式，包括手機 APP（但須保留版權聲明）

OFL 禁止的主要是明顯損害原作者著作權益的行為，例如拿原作者提供免費下載使用的 OFL 字體單獨販售，又或只是改變字體名稱就當成是自己出品等。留意本文不構成任何法律意見，請以官方授權原文為準。

關於昭源黑體可變式字型

思源黑體在 2021 年 4 月 8 日推出了 2.003 版更新，從這個版本開始正式支援 OpenType 可變式字型(Variable Font / VF) 格式。

「可變式字型」首見於 2016 年的 OpenType 字型規格。此格式容許將整個字型系列收納於單一檔案中，並提供多個變化軸（例如粗幼、闊窄），供使用者逐級調整。以思源黑體為例，在 VF 之前，字體提供 ExtraLight、Light、Normal、Regular、Medium、Bold、Heavy 七款固定字重。到了可變式字型，使用者就不必再受字體發佈時的固定層級所限制，而可以在最幼（細）的 ExtraLight（字重值 250）和最粗的 Heavy（字重值 900）兩個極端之間逐級調整，大大擴闊設計師的發揮空間和提升不同字體間的配搭彈性。

（圖片摘自 Adobe 關於思源黑體 VF 的新聞公報 )

昭源黑體標準版在 2.035 版更新中提供桌面和網頁字型的可變式字型格式，而昭源黑體 PRO 則在 1.005 版開始提供。使用者可以利用本網的線上預覽功能，一試昭源黑體 VF 在網頁使用時的效果。

OpenType 可變式字型的技術規格稱為 OpenType/CFF2，屬全新格式，與舊有的 OpenType 字體格式 (OpenType/CFF) 並不相容。雖然副檔名同樣是 .otf，但仍須在作業系統和應用程式更新、加入支援之後才能用到可變式字型的全部功能。某些作業系統或應用程式更可能會有兼容問題，例如已知在 Windows 10 (20H2) 系統雖然能辨認和安裝 OpenType/CFF2 可變式字型，但安裝後假如在 Microsoft Word（本身不支援可變式字型）嘗試選擇這款字體便會立即導致 Windows 系統字型渲染引擎崩潰、整個系統的字體無法顯示或出現異常而必須重新開機的情況。安裝使用前請先留意。

跟思源黑體一樣，昭源黑體桌面版的可變式字型是另外提供，安裝後的字型名稱是 Chiron Sans HK VF，跟原來的單字重 OTF 檔案沒有衝突，可以同時安裝。

網頁字型方面，未有可變式字型之前，網頁須為每個字重載入獨立的 CSS 和字型檔。轉用 VF 格式之後，使用時只須載入一個 CSS 檔案，即可在不同字重間切換。由於現在大多數瀏覽器已支援 WOFF2 可變式字型格式，而且絕大多數情況網頁使用字型時都不只需要存取一個字重，所以昭源黑體改為只發佈網頁字型 VF 格式，取代原來的單字重版本。

（更新日期：2021/5/4）

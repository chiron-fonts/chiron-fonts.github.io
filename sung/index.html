昭源宋體 (Chiron Sung HK) 是一款為繁體中文使用者而設、採用現代筆形風格、平衡標準字形和印刷體慣用筆形以適合作熒幕顯示和印刷用途的免費中文字體。

使用者可於此測試區輸入文字，並利用工具欄的選項，預覽昭源宋體在不同參數下的顯示效果。至於詳細説明和下載方法，請參看本字體的 Github 專案頁 (https://github.com/chiron-fonts/chiron-sung-hk)。

注意：
  1. 本網頁使用「昭源宋體網頁字型」排版，收字比桌面用的 OTF/TTF 版少，不能完全反映 OTF/TTF 版的收字量。
  2. 在文字框輸入文字後，瀏覽器或須下載相應的網頁字型子集以作顯示，這會導致額外的數據用量。

安裝前請先將所下載的發佈檔案（.zip 或 .tar.gz 副檔名）解壓縮。解壓縮後的子目錄存放了不同格式的字型檔案。目錄結構如下：

目錄內容

WOFF2_TTF/網頁字型 (TrueType 外框)

桌面用 OTF (OpenType) 字型

Section titled “桌面用 OTF (OpenType) 字型”

STATIC_OTF 或 OTF/ 載有 OpenType/CFF 格式的 OTF 字型，副檔名為 .otf，可以安裝在現時大多數桌面作業系統。各檔案所代表之字型屬性可從其檔名得知：

字體檔案命名方式字體家族名稱
昭源宋體ChironSungHK(-Text)?-[字重代號縮寫](-It)?.otfChiron Sung HK 或 Chiron Sung HK Text
昭源黑體ChironHeiHK(-Text)?-[字重代號縮寫](-It)?.otfChiron Hei HK 或 Chiron Hei HK Text
昭源環方ChironGoRoundTC-[字重值][字重代號縮寫].otfChiron GoRound TC

（“?” 代表前方括號內的部份可有可無。）

  • “Text” 代表該字型檔屬於 “Text” 分支家族。“Text” 家族的中文字字面較小，用在內文字體可以減低擁擠感。技術細節：Text 分支和新增字重頁有進一步説明。
  • “It” 代表該字型檔屬於「斜體」，沒有則是屬於「正體」。昭源環方只有正體，沒有斜體。

字重名稱反映的是該字型的粗幼（粗細）。OpenType 和 TrueType 格式字型會使用字重代號縮寫，例如 EL、L。

備註

採用縮寫主要是為了確保與各作業系統（主要是 Windows，Fontlab 也有説明文件指出若要與 Microsoft Office 兼容字體家族名稱須在 31 字元或以下）的兼容性。慣常做法是將字體的名稱總長度壓縮在 30 個字元以下。

備註：

  • 某些應用程式（例如 Microsoft Word）不會出現獨立的 Bold 字重選擇，而須使用者先選擇 Regular 字重、再透過該程式的「粗體」功能調出。「斜體」同理。這類程式的 Regular 字重一般不會在字型清單上顯示字重代號 “R” 而只會顯示字體家族名稱，其他字重則會有字重代號。

桌面用 TTF (TrueType) 字型

Section titled “桌面用 TTF (TrueType) 字型”

STATIC_TTF 或 TTF/ 目錄內放有 TrueType 格式字型，副檔名是 .ttf，其餘命名方式、安裝後的字型名稱等跟 OpenType 格式相同。由於 TrueType 和 OpenType 格式的系統字型名稱完全一樣，所以安裝時只能在 TrueType 或 OpenType 格式間二擇其一。

桌面用可變式字型 (Variable Font)

Section titled “桌面用可變式字型 (Variable Font)”

VAR/（兩種外框格式放於同一目錄）或 VAR_OTF/ 和 VAR_TTF/ （兩種外框格式分置不同目錄）內收錄的是昭源字體的可變式字型 (Variable Font, VF)。宋體和黑體除了提供「正體」，還有檔名有 It 的「斜體」。VF 格式可對字體參數進行更細緻的調整。

可變式字型安裝後的字體家族名稱如下:

字體字體家族名稱
昭源宋體Chiron Sung HK VF
昭源黑體Chiron Hei HK VF
昭源環方Chiron GoRound TC VF

VF 版比 OTF/TTF 版本多了一個 VF，所以可以跟 OTF/TTF 版本同時安裝。

昭源字體 VF 字型檔定義了以下變化軸：

  • Weight (wght)：字重軸，數值在 200 至 900 之間。
  • Padding (PADG) （限宋體、黑體）：字面變化軸，數值在 0 至 10 之間，預設值 0 代表無縮小，由 1 開始字面縮小比率逐級增加 0.5%，最大值為 10，即縮小 5%。OTF/TTF 字型檔的 “Text” 分支縮放率等同本軸的 “6”，即縮小 3%。

可變式字型規格在 2016 年 9 月發表，仍屬新技術，必須靠作業系統和軟件逐步提供支援，方能發揮最大效用。Windows 系統要到 Windows 11 23H2 (2023 年 11 月釋出) 版本才對 VF 字體提供基本支援。所謂「基本支援」，意思是沒有正式對 VF 字體作支援的軟件仍可以使用 VF 字體，但只能用到字體家族本身定義好的款式（以本字體而言是八個字重加兩個字面率），而不能像 Photoshop 這類有添加 VF 支援的軟件般對字體變化軸進行逐級微調。在 Windows 11 23H2 之前，即使安裝了昭源 VF 字體，不支援 VF 的軟件也只能用到家族的第一個款式（ExtraLight 非 Text 版）。

警吿

使用 Windows 10 或 Windows 11 (23H2 之前版本) 的使用者請勿嘗試安裝 OpenType/CFF2 格式（副檔名為 .otf）的 VF 字型檔。雖然可以成功安裝，但只要在個別軟件（例如記事本）選擇此一格式的 VF 字體，便會立即出現字型渲染引擎崩潰、畫面文字顯示異常、作業系統不穩而必須重新開機的致命性錯誤。.ttf 版本雖無此問題，但正如前述，23H2 之前的 Windows 11 對 VF 的支援程度只是非常初階。

備註：

  • 非官方定義的變化軸名稱須用全大寫，跟官方變化軸作區分，因此字重軸和字面調整軸名稱才會有大小寫的不同。
  • 又因 VF 格式不受之前 OTF、TTF 格式的字型名稱長度限制，所以字重名稱會採用長寫，不作縮略。不過作業系統為保持兼容性，或會自行裁切字體名稱。例如在 Windows 11 23H2 的 Microsoft Office 中，昭源黑體和宋體的 ExtraLight 字重 Text 版名稱分別是 “Chiron Hei HK VF ExtraLight Tex” 和 “Chiron Sung HK VF ExtraLight Te”，明顯是將第 31 個字元之後的 “t” 和 “xt” 截去。這並非字體出錯（在 Windows 11 個人化設定的字體頁面所顯示的款式名稱完全正常），而是作業系統為保持與原有 API 兼容性所致。

網頁字型

Section titled “網頁字型”

WOFF2_OTF/ 和 WOFF2_TTF/ 目錄下收錄昭源字體的網頁字型，供網頁開發人員使用。有關使用詳情請參閲網頁字型頁。

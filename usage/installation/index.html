取得字型

下載

Section titled “下載”

昭源宋體、昭源黑體均為開源字體，字型檔案可於 Github 取得。使用者可按下方之「下載」按鈕直接下載，或逕往相關的 Github 檔案庫下載。

網頁開發人員如想在網頁上使用昭源字體也可選擇使用 npm 套件或 CDN 服務。詳情請參閲網頁字型頁。

使用

Section titled “使用”

下載後請按照所使用作業系統的步驟，將字型安裝至系統中。視乎所使用的格式，字型名稱或會有所不同。詳情請參閲字型格式頁。
